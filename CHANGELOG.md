# Changelog

## [1.0.0] - 2025-12-02

### Added
- Initial release of Buddy Smart Daily Planner
- Activity management (create, edit, delete, track)
- Activity status tracking (done, missed, partial, rescheduled)
- Weather integration with OpenWeatherMap
- News feed integration with NewsAPI
- Stock market tracking with Alpha Vantage
- Weekly analytics and insights
- CLI tool for command-line usage
- React frontend with Tailwind CSS
- FastAPI backend with SQLAlchemy ORM

### Features
- Smart suggestions based on weather and patterns
- Local settings storage
- Weekly completion rate visualization
- Real-time data from multiple APIs
